Client Talking Points

Use concise definitions while showing live evidence.

Brute force: automated password guessing until a weak credential works.

SQL injection: attacker input changes the database query the app intended to run.

Session hijacking: stolen cookies are reused to impersonate a signed-in user.

Privilege escalation: a normal user gains administrative capability.

Log injection: forged log lines hide or confuse attacker activity.

Exfiltration: sensitive records are copied out of the application boundary.
